# 项目名称

## 环境配置

按照以下步骤配置和激活Python环境，并安装项目依赖：

1. 创建一个新的conda环境，并指定Python版本为3.9：
    ```bash
    conda create -n myenv python=3.9 -y
    ```

2. 激活刚创建的conda环境：
    ```bash
    conda activate myenv
    ```

3. 安装项目所需的依赖包：
    ```bash
    pip install -r requirements.txt
    ```

完成上述步骤后，你的环境应该已经配置好，可以开始使用项目了。
